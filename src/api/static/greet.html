¡Bienvenido!

Desde http:/localhost/6060/get_data puedes descargar los datos del proyecto con el token correspondiente.

Ejemplo de petición:

http://localhost:6060/get_data?eltoken=Z96474093

El archivo json disponible contiene 3 json:

json_dataset: contiene la lista de archivos y sus características usados para los modelos.

json_modelos: contienen la lista de modelos entrenados con todas sus características para poder emularlos

json_funcion_mysq: contien el codigo de la función que se utilizó para guardar los dataframe en la base de datos mysql

Desde http:/localhost/6060/predict puedes hacer una predicción de una fotografía que tengas

Te devuelve un json con la predicción y su probabilidad de acierto
